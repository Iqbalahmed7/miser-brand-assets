# 05 — Voice & tone

Miser reads like a well-edited broadsheet, not a landing page. Dry,
precise, slightly British, never preachy.

## Voice

- **Editorial.** Sentences are complete, clauses are earned. We use
  semicolons.
- **Understated.** We state facts; we don't sell them. If the fact is
  strong, it doesn't need help.
- **Dry.** One wry aside per page, usually in italics. Never a joke, often
  a raised eyebrow.
- **Confident, not boastful.** "Miser saved $4.39 today." Not "Miser can
  help you unlock savings."
- **Arithmetic-first.** Where a number exists, lead with it.
- **Universal.** Reads the same in London, Mumbai, Singapore. No idiom that
  requires cultural fluency to land.

## Tone ladder

| Context                    | Tone                                          |
| -------------------------- | --------------------------------------------- |
| Landing page hero          | Quiet defence. One italic, one number.        |
| Product UI · neutral state | Matter-of-fact. "Today's ledger."             |
| Product UI · savings       | Understated pride. "Saved today: $4.39."      |
| Product UI · failure       | Honest, calm. "This run failed. We kept the cache." |
| Marketing email            | Editorial. Write as if the reader is busy.    |
| Docs                       | Precise, patient, no jokes.                   |
| Social / launch            | Short, dry, one sentence maximum.             |

## Copy patterns

### Headlines

Use a phrase in italic to carry the idea. The italic is coloured greenback.

- *A quiet defence of the small number.*
- Built on *strict arithmetic*.
- Works at every size. *Especially the small ones.*
- Your agent has *expensive taste.*

### Asides

Drop in an italic, ink-3 aside to soften a claim with quiet self-awareness.
One per page, not per paragraph.

- "Miser is not, on balance, a lot." *(following a claim about savings)*
- "We're told this is boring. We take it as a compliment."
- "Nothing surprising. That's the point."

### Numbers

Always monospace. Always signed when appropriate. Never rounded away the
small cents — the small cents are the product.

- `Saved today: $4.39`
- `-$1.14 · routed opus → haiku`
- `Cache hit · 87%`

### Microcopy

Short, specific, never cute.

- Button: "Install Miser" *(not "Get started for free")*
- Link: "See the ledger" *(not "Learn more")*
- Empty state: "No runs today. No savings to report."
- Error: "Couldn't reach the API. We've paused." *(not "Oh no, something went wrong")*

## Do & don't

| ✓ Do                                             | ✗ Don't                                         |
| ------------------------------------------------ | ----------------------------------------------- |
| "Saved today: $4.39"                             | "Unlock savings with AI-powered optimisation"   |
| "Miser downgraded this run to Haiku."            | "We automagically chose the perfect model!"     |
| "Built on strict arithmetic."                    | "Built on powerful AI."                         |
| *Dry aside in italics, once per page*            | *Jokes, puns, exclamation marks*                |
| "A quiet defence of the small number."           | "Never overpay for AI again!!!"                 |
| Em dashes, semicolons, proper punctuation        | Slashes, ampersands, marketing-ese              |

## The one-sentence test

If a sentence wouldn't appear in *The Economist*, it doesn't belong on a
Miser surface. Rewrite it.
